# 智能体：理论基础与应用实践
该仓库为书籍《智能体：理论基础与应用实践》的配套开源仓库，主要包含1000+学术论文（基于智能体的框架、记忆、RAG、规划、行动、评估等核心模块分类汇总）以及相关代码（涵盖书中给出的代码示例以及有代表性的业界优秀开源代码链接参考），力求构建最全、最新、最准的智能体开源仓库，期待大家一起来建设该开源仓库。

## 内容简介
本书全面系统地介绍了智能体的理论基础与应用实践，内容从结构上可分为三部分。第一部分（第1~3章）系统梳理了智能体前置知识，包括智能体基础知识、自然语言处理基础知识及大模型基础知识。第二部分（第4~11章）是全书的核心内容，全面深入地剖析了智能体技术方法，涵盖智能体系统、感知能力、角色扮演能力、记忆能力、检索增强生成、规划能力、执行能力、智能体评估，以及智能体技能等内容。各章不仅阐述了各种能力的核心原理，还同步提供了代表性技术方法，帮助读者快速掌握并落地使用。第三部分（第12~15章）从应用实践角度，全面梳理了智能体开发相关工具，并通过Manus智能体剖析和DeepSeek开发实践两个章节，详细介绍了从实现思路到完整代码的全流程，使读者能够学以致用。
本书内容兼具系统性与实用性，既可作为智能体领域从业者的进阶学习用书，也适合对智能体感兴趣的初学者入门参考。同时，本书还可作为高等院校计算机科学与技术、人工智能、自动化及相关专业的师生用书，以及培训学校的教材。

## 前沿/序言 （芮勇博士）
智能体，让人工智能从“对话”走向“行动”
2022年11月，ChatGPT横空出世，展现了大模型令人惊叹的能力，仿佛人工智能的“奇点时刻”就在眼前。然而，作为在这一领域深耕三十载的研究者与实践者，我清醒地认识到大模型虽强大，却并非万能。如今，智能体的发展，正在为我们提供一条推动大模型从“对话工具”逐步进化为“行动伙伴”的重要路径。当然，这一路径仍处于探索阶段，智能体并非解决所有问题的万能钥匙，但其方向无疑值得期待。《智能体：理论基础与应用实践》一书的推出恰逢其时，相信该书能够为学术界与产业界提供一份有价值的理论指导与实践参考。
尽管大模型在自然语言理解和生成方面表现出色，但其在实际应用中仍面临诸多挑战。知识时效性是首要问题，大模型的知识截止于训练数据的时间点，无法获取最新信息，这对其应用于需要实时动态信息的业务场景形成制约。此外，大模型输出具有概率性和不确定性，导致其在实际应用中的可靠性与一致性面临较大挑战。更为关键的是，大模型只能“被动响应”用户提问，缺乏规划与执行能力，无法主动分解复杂任务并持续执行，更无法调用外部工具或根据反馈动态调整。而且，大模型参数量动辄达数千亿级别，对计算资源的需求巨大，部署受限，且生成的内容难以溯源，缺乏可解释性，在医疗、金融等对准确性以及可解释性要求高的领域落地困难。
智能体的出现，标志着人工智能从“被动响应”向“主动服务”迈出了重要一步。OpenAI提出的人工智能五级分类体系清晰地勾勒出这一演进路径：第一级是聊天机器人，第二级是推理者，第三级便是智能体（能够代表用户执行操作的智能系统）。智能体的核心特征是“自主性”，即无须过多的人类干预，便能独立感知环境、制订计划、执行操作并持续优化。真正的智能不仅在于单点能力的强大，更在于如何将感知、记忆、规划和执行有机集成，形成闭环的智能系统。
智能体通过其独特的架构设计，系统性地弥补了大模型的不足。感知模块赋予智能体环境感知能力，使其能够实时获取外部信息，突破知识截止日期的限制。记忆系统让智能体具备短期和长期记忆能力，不仅能记住对话历史，还能积累经验、沉淀知识，实现持续学习和能力迭代。规划引擎作为智能体的“大脑中枢”，能够将复杂任务分解为可执行的子任务序列，并根据执行反馈动态调整策略。工具调用能力让智能体可以主动调用各种API、数据库、计算工具乃至物理设备，将大模型的语言理解能力转化为在真实世界中的执行力。反馈与自我优化机制则让智能体具备从经验中学习的能力，不断优化决策策略和工作流，展现出较强的适应性和进化潜力。
值得强调的是，智能体并非要替代大模型，而是将大模型作为其“认知核心”，通过架构创新充分释放大模型的潜力。智能体正是在“有大模型”的基础上，通过系统化设计实现对大模型能力的延展与增强。这种设计使智能体能够在人类设定目标后承担更多执行工作，但在关键决策点仍需要人类确认和介入，这也是当前技术发展阶段的务实选择。
《智能体：理论基础与应用实践》一书的价值不仅在于对智能体技术的理论阐释，更在于为实践者提供了较为完整的方法论和应用指导。全书分为三部分，构建了从智能体前置知识到核心技术，再到应用实践的知识体系。书中对智能体核心组件的深入剖析尤为可贵，特别是第11章中对智能体安全评估的专门讨论，体现了作者对技术伦理的重视。书中指出，智能体作为依托大模型的集成性系统，不仅继承了大模型从数据准备、预训练到部署全生命周期中的数据隐私和模型漏洞等固有安全风险，还可能引入与外部模块交互过程中产生的记忆信息泄露、恶意工具调用等新型安全挑战。在当今人工智能技术快速发展的背景下，我们不仅要追求技术的突破，更要关注其对社会的影响。我一直认为，真正的智能不仅体现在能力上，更体现在责任与伦理上。
第15章对电子商务、教育培训、医疗健康、金融服务、智能制造等领域的应用分析，展现了智能体的广阔前景。这些案例不仅是对技术应用的展示，还揭示了智能体如何重塑产业形态、创造新的商业价值。从联想的实践经验来看，智能体正在成为企业数字化转型的重要使能技术。书中对智能体发展趋势的判断，如“智能体的具身化程度更高”“智能体人格化特性更显著”等观点，既具前瞻性又务实可行。
从2016年AlphaGo战胜李世石，标志着人工智能在特定领域超越人类，到2022年ChatGPT展现大模型的强大能力，再到如今智能体的蓬勃发展，人工智能正经历着从特定领域到通用能力、从被动响应到主动服务的渐进式演进。近年来，诺贝尔奖对人工智能的认可，全球科技巨头对智能体的战略布局与重金投入，都预示着这一领域蕴含着巨大的发展潜力。当然，从当前的智能体到真正成熟的“行动伙伴”，还有很长的路要走，这也为研究者和实践者留下了广阔的探索空间。
对任何希望学习与实践智能体的研究者、工程师、企业决策者而言，这本书都是不可多得的参考指南。它既适合学术界阅读参考，也适合产业界落地实践。通过阅读本书，读者不仅能掌握智能体的原理，还能理解其在推动产业变革中的作用。我相信，《智能体：理论基础与应用实践》的出版，能够吸引更多研究者和实践者投身到智能体的研究与应用中，共同推动大模型从“对话工具”逐步进化为“行动伙伴”，让人工智能具备从理解到执行的系统性能力。
